# 技术设计: 量子未来体验与内容智能扩展

## 技术方案
### 核心技术
- React 18 + Vite 6
- styled-components + CSS Variables
- Framer Motion
- 本地存储与本地计算（无后端依赖）

### 实现要点
- 视觉层：重构 Design Tokens，统一未来感配色、背景网格、霓虹层级
- 业务层：新增内容洞察计算（标签趋势 / 工作室雷达 / 口碑脉冲）
- 体验层：追更计划器 + 推荐匹配度解释 + 详情页口碑摘要
- 性能层：减少重复渲染、缓存计算、避免不必要的 DOM 变更

## 架构设计
无新增系统级架构变更，沿用现有 App Shell + Local-first 数据层。

## 架构决策 ADR
### ADR-20260111-01: 本地内容洞察引擎
**上下文:** 需要新增多维度内容洞察与匹配度解释
**决策:** 基于现有数据模型与本地计算生成 Tag/Studio/Audience 指标
**理由:** 无后端依赖、响应快、可离线
**替代方案:** 引入远程推荐服务 → 拒绝原因: 违背本地优先与静态站架构
**影响:** 新增 utils 模块与局部 UI 组件
