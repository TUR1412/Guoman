# HelloAGENTS 知识库入口

> 本目录作为本项目的知识库（SSOT），用于沉淀：项目上下文、模块说明、方案包、归档与变更记录。

## 快速入口

- 项目概览：`README.md` / `README.en.md`
- 架构与规范：`docs/ARCHITECTURE.md` / `docs/DESIGN_TOKENS.md`
- 诊断与性能：`docs/DIAGNOSTICS.md` / `docs/LIGHTHOUSE_BASELINE.md`
- 迭代与审计：`docs/ITERATIONS.md` / `docs/QUARK_AUDIT.md`

## 知识库文件

- 项目上下文：`helloagents/context.md`
- 技术约定：`helloagents/project.md`
- 变更记录（知识库维度）：`helloagents/CHANGELOG.md`

## 知识库结构

- 方案包：`helloagents/plan/`
- 模块文档：`helloagents/modules/`
- 已归档方案包：`helloagents/archive/`
- 其他资料：`helloagents/wiki/` / `helloagents/history/`
