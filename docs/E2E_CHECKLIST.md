# E2E 测试清单（手动）

> 目标：覆盖核心路径、空状态、导入导出与数据持久化。

## 1. 首页

- 轮播可以切换
- 继续观看 / 最近浏览空状态引导正常

## 2. 作品详情

- 观看进度可修改并保存
- 收藏、分享、生成海报按钮可用
- 评论可发布、清空

## 3. 搜索

- 多关键词搜索可用
- 搜索历史可记录/清空

## 4. 收藏

- 收藏导入/导出
- 分组创建、分配、删除

## 5. 排行榜 / 推荐 / 资讯

- 标签切换可用
- 进入资讯详情并返回正常

## 6. 用户中心

- 数据导入导出可用
- 快捷键开关生效
- 通知与反馈可新增/清空

## 7. 无障碍

- Tab 可遍历关键交互
- Skip link 能跳到主内容

## 8. 兼容性

- 深浅主题切换
- 低网速下页面仍可加载
